# PhoneBridge v1.0 Release Notes

**PhoneBridge v1.0 现已正式发布！🎉**

这是首个稳定版本，我们实现了轻量级的 iOS 与 Windows 局域网无缝协同，带来了以下核心体验升级：

## ✨ 新特性 (New Features)
- **极速无痕同步**：支持 iPhone 到 Windows 剪贴板的秒级文字覆盖，以及图片/文件的高速局域网直传。
- **系统托盘驻留**：完全后台化，彻底告别控制台黑框，安静驻留于 Windows 任务栏右下角。
- **开机自启能力**：支持一键在托盘菜单中开启/关闭 Windows 系统随机器自动启动。
- **动态网络寻址 (mDNS)**：自动释放 `_phonebridge._tcp` 广播，未来客户端可零配置即插即用寻找宿主机。
- **安全拦截防线**：所有 API 均接入 Bearer Token 强鉴权中间件，并内置存储路径穿越免疫机制，全面保障局域网数据隐私及文件安全。

## 🛠 技术细节 (Under the Hood)
- 采用原生 Go 1.22 强力驱动，配合 `net/http` 原生路由分发。
- C/S 极简架构，彻底去除冗余第三方大体积框架。
- 利用 Mutex 协程锁确保存储与系统调用层面的并发一致性。
- 已引入 `gorilla/websocket` 为 v2.0 双向消息推送与大模型端侧通信留出生态底座。

## 📦 如何使用
1. 在 [Releases](#) 页面下载 `phonebridge.exe` 并运行。
2. 按照 README 中的配置指南在 iOS 快捷指令中添加相应的动作。
3. 您可以搭配 iOS 系统的 "轻点背面" 或 "NFC 自动化" 获得无感零点击体验！
